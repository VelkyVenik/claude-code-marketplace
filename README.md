# Venik Claude Code Marketplace

A collection of Claude Code plugins for developer productivity.

## Installation

```bash
claude plugin marketplace add VelkyVenik/claude-code-marketplace
```

## Available Plugins

### workstation-bootstrap

Complete developer workstation provisioning for **macOS** and **Ubuntu**.

12-phase interactive setup: security hardening, Homebrew/apt, Ghostty, Fish shell, modern CLI tools, Vim, Git/GitHub CLI, Node.js/Python/Docker, tmux, SSH keys, and Claude Code status line.

```
/workstation-bootstrap:setup
```

[Full documentation](workstation-bootstrap/README.md)
